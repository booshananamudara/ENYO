EnyoCart — Icon Generator

Renders the ENYO logo at each required size and lets you download each PNG.

How to use

  1. Open this file in a browser (Chrome/Edge/Firefox).
  2. Click the Download button below each icon size to save the PNG.
  3. Move each downloaded PNG into the assets/icons/ folder:
    • icon-16.png
    • icon-32.png
    • icon-48.png
    • icon-128.png
  4. Load the extension via chrome://extensions → Load unpacked.
